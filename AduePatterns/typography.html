Notes :

使用 便捷CSS类 搭配，设置底部外边距（margin-bottom）大小。

便捷CSS类:

  • margin-bottom--lv0 for 0px margin-bottom
  • margin-bottom--lv1 for 6px margin-bottom
  • margin-bottom--lv2 for 12px margin-bottom
  • ......
  • margin-bottom--lv8 for 48px margin-bottom

段落

Notes :

<p>（段落）元素被设置了10px的底部外边距（margin）。使用 便捷CSS类 搭配，设置字体大小。

便捷 CSS 类:

  • small-meta for 13px text
  • x-small-meta for 12px text
  • xx-small-meta for 10px text
  • dim-el for light gray text (80% opacity)
