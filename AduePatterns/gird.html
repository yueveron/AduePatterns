分栏系统

  • 常规分栏
  • 混合分栏
  • 固定分栏

分栏系统由行（row）与列（column）组合构成。注意事项如下：

  • 通过为“列（column）”设置 padding 属性，从而创建列与列之间的间隔（gutter）。通过为 .row 元素设置负值 margin 从而抵消掉为 .container 元素设置的 padding( padding-right: 15px; padding-left: 15px; )，也就间接为“行（row）”所包含的“列（column）”抵消掉了padding。
  • .row 默认属性： margin-left: -15px; margin-right: -15px;
  • .col 默认属性：float: left; padding-right: 15px; padding-left: 15px;

混合分栏

混合分栏，通过混合以及嵌套，从而实现更丰富的分栏布局。分类如下：

1/8 与 7/8 组合

1 / 8

7 / 8

3/8 与 5/8 组合

3 / 8

5 / 8

1/4 与 3/4 组合

1 / 4

3 / 4

1/3 与 2/3 组合

1 / 3

2 / 3

1/3 与 2/3 组合 - 1/2 与 1/2 组合嵌套于 2/3 内

1 / 3

1 / 2

1 / 2

Notes :

左侧例子中，其余列使用的类名如下：

  • 1/8 与 7/8 组合 - col eight , col eight-7
  • 3/8 与 5/8 组合 - col eight-3 , col eight-5
  • 1/4 与 3/4 组合 - col fourth , col fourth-3
  • 1/3 与 2/3 组合 - col third , col third-2

使用嵌套时（如左侧例子中），必须使用 group 作为父层类标识。

固定分栏

固定分栏，当屏幕宽度小于或等于 767px，“列（column）” 的宽度保持固定不变，不会自动成为单列。

Notes :

固定分类，只需使用 static 作为 “列（column）” 标记即可。
