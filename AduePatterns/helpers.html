我们定义了以下常用的 CSS 类，使用它们能够快速地对元素进行样式设置，并且避免了创建过多的 CSS 类。

间距

为了保证间距 CSS 类命名的一致性及层级的递增性, 我们使用 level system 法则对 padding 和 margin 进行定义。 使用 .padding--lv1 到 padding--lv8 来定义 padding ，使用 .margin--lv1 到 margin--lv8 来定义 margin。对于间距为零的 padding 或者 margin，则使用 .padding--lv0 或 .margin--lv0 来定义

Level 1: 6px

Level 2: 12px

Level 3: 18px

Level 4:24px

Level 5: 30px

Level 6: 36px

Level 7: 42px

Level 8: 48px

Notes :

单个方位的零间距 padding 或者 margin , 请使用以下 Css Class：

  • Top 零间距：padding-top--lv0 或 margin-top--lv0
  • Bottom 零间距：padding-bottom--lv0 或 margin-bottom--lv0
  • Top-Bottom 零间距：padding-top-bottom--lv0 或 margin-top-bottom--lv0
  • Left 零间距：padding-left--lv0 或 margin-left--lv0
  • Right 零间距：padding-right--lv0 或 margin-right--lv0
  • Left-Right 零间距：padding-left-right--lv0 或 margin-left-right--lv0

以下是 margin-bottom 的层级 Css Class：

  • margin-bottom--lv1 ：margin bottom 6px
  • margin-bottom--lv2 ：margin bottom 12px
  • margin-bottom--lv3 ：margin bottom 18px
  • margin-bottom--lv4 ：margin bottom 24px
  • margin-bottom--lv5 ：margin bottom 30px
  • margin-bottom--lv6 ：margin bottom 36px
  • margin-bottom--lv7 ：margin bottom 42px
  • margin-bottom--lv8 ：margin bottom 48px
